# Bezalel

Bezalel is a redemptive-tech consultancy — a "ThoughtWorks for redemptive technology" — operating through [FaithTech Toronto](https://faithtech.com). We help churches, ministries, and mission-driven nonprofits build genuinely good technology with craftsmanship, not just good intentions.

This repository is not a software product. It is the home of Bezalel's idea, operating model, and playbooks — a living knowledge base for anyone deciding whether to build with us.

## The flywheel

1. **Create** — FaithTech Create programs develop practitioners and surface real problems from churches and ministries.
2. **Hackathons** — Teams prototype solutions against those real problems in a compressed, high-energy format.
3. **Bezalel engagements** — The most promising prototypes get taken from demo to durable, well-architected software by a Bezalel team.
4. **Feedback** — Outcomes, learnings, and newly-formed practitioners flow back into Create and the next hackathon cycle.

See [`docs/model/the-pipeline.md`](docs/model/the-pipeline.md) for the detailed hand-offs between stages.
