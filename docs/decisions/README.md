# Decisions

Lightweight ADR-style decision records for the Bezalel initiative itself — not for an individual partner project's technical decisions, but for changes to the model FaithTech Toronto is exploring.

## Format

Each decision is a numbered file: `NNNN-short-title.md`. Include context, the decision, and consequences. Once recorded, a decision is not edited to reflect later changes — a superseding decision is added instead, with a reference back.
